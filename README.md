# Circles Bounce Simulation
---
## Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск сервера разработки
npm run dev
```

Откройте браузер по адресу **http://localhost:5173/**

---

## Управление

Кнопка **Добавить круг** добавляет круг в рандомном месте с заданными параметрами
Кнопка **Очистка сцены** удаляет все круги со сцены
При нажатии на **Space** можно останавливать и запускать симуляцию

Сохранение и восстановление состояния реализовано через LocalStorage

## Архитектура
Сцена состоит из прямоугольника и кругов. Прямугольник и круг рисуются разными шейдерами(для всех кругов используется 1 программа).
Физика отскока реализована на процессоре с перепором всех кругов.

Есть 2 возможности оптимизации.
1. Использовать 1 геометрию для всех кругов, уменьшит потребляемую память и ускорит отрисовку.
2. Перенести вычисления на видеокарту c помощью Transform Feedback или перевести проект на WebGPU и симуляцию реализовать через ComputeShader.
